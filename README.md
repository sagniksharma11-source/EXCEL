**1. `excel_dashboard.xlsx` — TCS Financial Ratio Dashboard (FY21–FY25)**

This is a financial analysis project on **Tata Consultancy Services (TCS)**. It has two sheets — raw financial data (Sales, Net Profit, EPS, Borrowings, Assets) and a dashboard sheet with key ratios like ROE, ROCE, Net Profit Margin, Debt/Equity, P/E Ratio, and EPS Growth tracked across 5 years. The insights show TCS as a near debt-free, highly profitable company with expanding ROE and stable margins. A strong **equity research / valuation project** for your portfolio.

---

**2. `Frank_BAss_Diffusion_Model_dataset.xlsx` — Bass Diffusion Model**

This is a **marketing analytics / forecasting project** using the Bass Diffusion Model, which predicts how a new product gets adopted over time. It contains actual vs. predicted adopter data across 20 periods, with error and squared error calculations for model accuracy. It includes an innovator coefficient (p = 0.03) and imitator coefficient (q = 0.38), showing a classic bell-curve adoption pattern with ~4,900 total adopters.

---

**3. `Market_Basket_Dataset2_sagnik_sharma.xlsx` — Market Basket Analysis**

This is a **retail analytics project** using Association Rule Mining (Apriori logic). It has 15 grocery transactions with 6 items (Rice, Dal, Oil, Salt, Soap, Shampoo), a binary matrix for item presence, and an Excel Solver model to find the most frequently co-purchased item combinations. Rice is the most popular item (9/15 transactions), and the Solver is set up to maximize support count subject to basket size constraints.
